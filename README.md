# Profile Page

A simple web project for creating and displaying user profile information.

## Features

- Responsive design
- Editable user details
- Profile picture upload
- Basic styling

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2. Open `index.html` in your browser.

